
# 🧠 NeuroRoute AI Platform
# Deployment Link :-
🚀 **Predict. Optimize. Deliver Smarter.**

NeuroRoute AI Platform is an **AI-powered logistics intelligence system** that predicts delivery time using machine learning and provides actionable insights for route optimization.

---

## 📌 Overview

This project is a **Streamlit-based interactive dashboard** that uses a **Linear Regression model** to estimate delivery time based on multiple real-world factors such as:

* 📍 Distance
* 🌦 Weather Conditions
* 🚦 Traffic Level
* 🕒 Time of Day
* 🚚 Vehicle Type
* ⏱ Preparation Time
* 🧑‍💼 Courier Experience

---

## ⚙️ Features

✅ Real-time delivery time prediction
✅ Interactive UI with modern dashboard design
✅ Route & delivery analytics
✅ Traffic impact simulation
✅ Preparation time sensitivity analysis
✅ Clean and responsive Streamlit interface

## 🧠 Machine Learning

* **Algorithm:** Linear Regression

```

```
